Here are some of my projects

  • Apple Web Clone

    React, Vite, Three.js, GSAP, Tailwind.css

    This Stunning Apple Clone utilized the React + Vite lightweight combo to maintain extremely high performance and load times, even when packaged with high resolution and ultra dynamic iPhone 15 Model Animations. All the thrilling animation functionality was brought to life with GSAP and Three.js, with Tailwind.css responsible for the rest of the UI design.

  • Brainwave AI

    React, Vite, JavaScript, Tailwind.css

    This Robust and attractively designed project utilized Tailwind.css to create gorgeous gradients, an easy-on-the-eyes color theme, and Modern Bento-Box UI design components, all working together to produce a website that maintains user visit duration above the industry average of 53 seconds.

  • The Collie Kennel

    React, Vite, Javascript, Tailwind.css

    Inspired by the increasing number of Border Collies(BCs) ending up in shelters, I designed a mock 3rd-party adoption site that connects prospective adopters with BCs in shelters all across the US! with a mock API and personally created data set, I combined them to form the dynamic BC list as well as each BC's profile, complete with skeleton loading states.

  • NFT Marketplace

    React, React-Slick Slider, AOS Animations, Javascript, Tailwind.css

    This internship project was completed utilizing Git Branching strategies that paralleled a real team production environment. I worked on multiple high level component functions, including carousels, real-time countdown timers, dynamic API-powered routing and pages, API filtering and slicing, etc. Each component was designed with User Experience in mind, with skeleton loading states and animations from the AOS library.

  • E-Commerce Library

    React, JavaScript, HTML, CSS

    This digital Library utilizes React-Router-Dom to power dynamic routes and flawless API integration for dynamic data fetching and rendering. This website also features the ability to sort books by filters, add items to your cart while updating the items and total, and conditionally rendering an empty cart display.

  • Nike Web Clone

    React, Vite, JavaScript, Tailwind.css

    This Nike Storefront is a Tailwind.css centric page, and focuses on displaying the ability of Tailwind to transform websites from basic to brilliant in a fraction of the time of bare CSS. The page also features clickable shoe options to change part of the page; feel free to try it out!
